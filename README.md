
# Rent Invest - Real Estate Management Website

## Project Overview

This project is a real estate management website for Rent Invest, a company specializing in property management and investment services in Wroc≈Çaw, Poland. The website serves as a digital business card and presents the company's services to potential clients.

## Features

- Modern, responsive design
- Clear service presentation for different client segments
- Interactive components including:
  - Client testimonials carousel
  - Service path selection
  - Contact form
  - Interactive location map
- Multiple pages including Home, About Us, and Contact

## Tech Stack

This project is built with:
- React
- TypeScript
- Tailwind CSS
- React Router DOM for navigation
- Shadcn UI for UI components

## Local Development

To run this project locally:

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

## Project Structure

- `src/components`: Reusable UI components
- `src/pages`: Page components that make up the routes
- `src/hooks`: Custom React hooks
- `public`: Static assets
